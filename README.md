# Dota 2 Counterpicker 🎮

[English](#english) | [Русский](#русский)

---

## English

A lightweight Python desktop application designed to help players quickly choose effective counter-picks against enemy heroes during the Dota 2 drafting phase.

### ✨ Features
* **User-Friendly GUI:** Fully built using the **Tkinter** library.
* **Fast & Lightweight:** Starts instantly and doesn't distract you from the game.
* **Local Execution:** Runs entirely on your machine without complex setup.

### 🛠️ Tech Stack
* **Python 3.x**
* **Tkinter** (built-in GUI library)

### 🚀 How to Run Locally

1. Make sure you have **Python 3** installed.
2. Clone this repository or just download the project files.
3. Open your terminal in the project folder and run:
   ```bash
   python WindowTest2.py
   ```

## Русский

Легкое десктопное приложение на Python, которое помогает быстро подобрать эффективные контрпики против героев соперника прямо во время стадии драфта в Dota 2.

### ✨ Особенности
* **Удобный графический интерфейс:** Полностью реализован на библиотеке **Tkinter**.
* **Простота и скорость:** Приложение запускается мгновенно и не нагружает систему.
* **Локальная работа:** Все скрипты работают прямо на твоем компьютере без лишних настроек.

### 🛠️ Стек технологий
* **Python 3.x**
* **Tkinter** (встроенная библиотека для GUI)

### 🚀 Как запустить проект локально

1. Убедись, что на компьютере установлен **Python 3**.
2. Склонируй этот репозиторий или просто скачай файл проекта.
3. Открой терминал в папке с проектом и запусти главный файл:
   ```bash
   python WindowTest2.py
   ```
